# Personal Portfolio Concept

A simple, modern, and sleek personal portfolio website template. Built using HTML, CSS, and basic Vanilla JavaScript. Features a dark theme, custom properties, glassmorphism UI elements, smooth scrolling, and scroll-triggered animations.

## Features
- **Responsive Layout:** Works on mobile and desktop devices.
- **Glassmorphism:** Navigation and cards utilize blurring and transparency for a premium look.
- **Dynamic Animations:** Hero type-writer effect, fading reveal animations, and hover-lift cards.
- **Dark UI:** Beautiful custom dark theme with vibrant neon accents.

## How to use
- Clone the repository.
- Open `index.html` in your web browser. Or serve the directory using a local web server (e.g. `python -m http.server 8000`).
